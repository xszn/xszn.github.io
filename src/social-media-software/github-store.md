# Github Store

## 介绍

<Img alt="“Github Store”屏幕截图" src="/images/social-media-software/github-store/screenshot.webp" />

Github Store 是一款应用程序商店。在应用内浏览、发现并一键安装来自 GitHub Releases 的应用，让用户可以方便地跨平台发现、安装、跟踪和更新这些应用，类似于 F-Droid 或其他应用商店概念，但直接集成在代码托管平台。

## 系统平台

Android、iOS、Windows、Linux

## 下载

网站：无

源码：https://github.com/rainxchzed/Github-Store
