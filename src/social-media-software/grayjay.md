# Grayjay

## 介绍

<Img alt="“Grayjay”屏幕截图" src="/images/social-media-software/grayjay/screenshot.webp" />

Grayjay 是一款视频聚合器。将来自 YouTube、PeerTube、Twitch 等多个平台的视频内容整合到一个应用程序中，无需在不同平台之间切换。

## 系统平台

Android、Windows、macOS、Linux
