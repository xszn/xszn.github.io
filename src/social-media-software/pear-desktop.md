# Pear Desktop

## 介绍

<Img alt="“Pear Desktop”屏幕截图" src="/images/social-media-software/pear-desktop/screenshot.webp" />

Pear Desktop 是一款 YouTube Music 客户端。提供了更接近原生应用的界面、功能和完整的桌面音乐播放体验，并具备插件、主题和翻译等丰富扩展性。

## 系统平台

Windows、macOS、Linux
