# PeaZip

## 介绍

压缩软件。除了支持自己原生的 PEA 压缩格式（主要功能为压缩、分档、弹性地认证加密、集成式检查架构）外，还有支持其他的主流压缩格式。不过有特别着重在处理开放许可的压缩格式上。
