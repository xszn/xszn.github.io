# PeaZip

## 介绍

<Img alt="“PeaZip”屏幕截图" src="/images/foss/system-software/peazip/screenshot.webp" />

PeaZip 是一款压缩软件。除了支持自己原生的 PEA 压缩格式（主要功能为压缩、分档、弹性地认证加密、集成式检查架构）外，还有支持其他的主流压缩格式。不过有特别着重在处理开放许可的压缩格式上。

## 系统平台

Windows、macOS、Linux

源码：https://github.com/peazip/PeaZip
