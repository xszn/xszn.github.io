# BleachBit

## 介绍

磁盘清理工具。清理文件以释放磁盘空间并保护隐私。专为 Linux 和 Windows 系统设计，可以彻底清除数千个应用程序，包括 Firefox、Adobe Flash、Google Chrome、Opera 等。
