# BleachBit

## 介绍

<Img alt="“BleachBit”屏幕截图" src="/images/foss/system-software/bleachbit/screenshot.webp" />

BleachBit 是一款磁盘清理工具。可以清理文件，释放磁盘空间并保护隐私。只需勾选相应的复选框，即可删除不必要的文件，例如缓存、日志、Cookie、最近使用的文件列表等。

## 系统平台

Windows、Linux
