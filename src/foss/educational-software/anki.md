# Anki

## 介绍

抽认卡软件。全球最受欢迎的间隔重复学习工具之一。主要用于制作和学习抽认卡。该软件基于认知科学研究，利用间隔重复和主动回忆的原理来提高学习效率和长期记忆保持。
