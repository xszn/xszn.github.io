# Handy

## 介绍

<Img alt="“Handy”屏幕截图" src="/images/foss/productivity-software/handy/screenshot.webp" />

Handy 是一款语音转文本软件。提供简单易用且注重隐私的语音转录功能。只需按下快捷键，说出想表达的内容，即可将其显示在任意文本框中，完全离线可用。

## 系统平台

Windows、macOS、Linux
