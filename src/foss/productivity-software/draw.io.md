# draw.io

## 介绍

<Img alt="“draw.io”屏幕截图" src="/images/foss/productivity-software/draw.io/screenshot.webp" />

draw.io 是一款流程图软件。支持绘制流程图、UML 图、网络图等多种类型的图表。可以通过在线编辑器，连接各种云存储平台，也可下载独立桌面应用程序以供离线使用。

## 系统平台

Web、Windows、macOS、Linux

## 下载

网站：https://www.drawio.com/

源码：https://github.com/jgraph/drawio-desktop
