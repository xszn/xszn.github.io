# OBS Studio

## 介绍

<Img alt="“OBS Studio”屏幕截图" src="/images/foss/video-software/obs-studio/screenshot.webp" />

OBS Studio 是一款屏幕录制软件。用于直播和屏幕录制。可以将视频流传输到 Twitch、YouTube 和其他众多平台，也可以使用高质量的 H264/AAC 编码录制自己的视频。

## 系统平台

Windows、macOS、Linux

## 下载

网站：https://obsproject.com/

源码：https://github.com/obsproject/obs-studio
