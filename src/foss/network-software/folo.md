# Folo

## 介绍

<Img alt="“Folo”屏幕截图" src="/images/foss/network-software/folo/screenshot.webp" />

Folo 是一款RSS 阅读器。由 AI 驱动，具备上下文 AI，将内容整理成一个时间线，只专注于重要信息，远离信息噪音。分享列表、探索合集，享受无干扰的浏览体验。

## 系统平台

Web、Android、iOS、Windows、macOS、Linux

## 下载

网站：https://folo.is/

源码：https://github.com/RSSNext/Folo

## 使用教程

无。
