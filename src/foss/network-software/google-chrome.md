# Google Chrome

## 介绍

网页浏览器。全球使用最广泛的网页浏览器。采用 Blink 排版引擎，并使用高性能 JavaScript 引擎“V8”。其主要目标是提升浏览器的稳定性、速度和安全性，同时创造简洁高效的用户界面。
