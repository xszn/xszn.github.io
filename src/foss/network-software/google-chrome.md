# Google Chrome

## 介绍

<Img alt="“Google Chrome”屏幕截图" src="/images/foss/network-software/google-chrome/screenshot.webp" />

Google Chrome 是一款网页浏览器。是目前全球使用最广泛的网页浏览器。采用 Blink 排版引擎，并使用高性能 JavaScript 引擎“V8”。其主要目标是提升浏览器的稳定性、速度和安全性，同时创造简洁高效的用户界面。

## 系统平台

Android、iOS、Windows、macOS、Linux

## 下载

网站：https://www.google.com/chrome/

源码：无。

::: info
虽然 Chrome 本身并非自由软件且未开放源代码，但 Google 有相应的开放源代码计划，名为 [Chromium](https://www.chromium.org/)。
:::
