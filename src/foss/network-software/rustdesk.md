# RustDesk

## 介绍

<Img alt="“RustDesk”屏幕截图" src="/images/foss/network-software/rustdesk/screenshot.webp" />

RustDesk 是一款远程桌面软件。TeamViewer 或 AnyDesk 的开源替代品。主要由 Rust 编写，可以远程维护计算机和其他设备。无需依赖 VPN 或端口转发等额外工具，甚至在防火墙或 NAT 后面也能运行。

## 系统平台

Web、Android、iOS、Windows、macOS、Linux

## 下载

网站：https://rustdesk.com/

源码：https://github.com/rustdesk/rustdesk
