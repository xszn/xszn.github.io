# AB Download Manager

## 介绍

<Img alt="“AB Download Manager”屏幕截图" src="/images/foss/network-software/ab-download-manager/screenshot.webp" />

AB Download Manager 是一款下载管理器。通过与浏览器的无缝集成从任何地方轻松下载文件，享受快速、免费的下载体验。比以往更高效地管理和整理下载内容。

## 系统平台

Windows、macOS、Linux
