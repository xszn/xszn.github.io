# LocalSend

## 介绍

<Img alt="“LocalSend”屏幕截图" src="/images/foss/network-software/localsend/screenshot.webp" />

LocalSend 是一款文件共享软件。将文件分享到附近的设备。在本地网络上安全地与附近设备分享文件和消息，无需互联网连接。

## 系统平台

Web、Android、iOS、Windows、macOS、Linux
