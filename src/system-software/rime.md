# Rime

## 介绍

<Img alt="“Rime”屏幕截图" src="/images/system-software/rime/screenshot.webp" />

Rime 是一款输入法引擎。也是一个跨平台的输入法算法框架。基于这一框架，开发者与其他开源社区的参与者在各系统平台上创造了不同的输入法前端实现。

::: info 相关推荐
[雾凇拼音](https://dvel.me/posts/rime-ice/)：是 Rime 的一份配置仓库。提供了一套开箱即用的完整配置，包含输入方案（全拼、常见双拼）、长期维护的开源词库及各项扩展功能。
:::
