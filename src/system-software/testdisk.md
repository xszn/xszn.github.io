# TestDisk

## 介绍

<Img alt="“TestDisk”屏幕截图" src="/images/system-software/testdisk/screenshot.webp" />

TestDisk 是一款数据恢复软件。主要用于在分区表损坏、误删分区、病毒感染导致无法启动等情况下，恢复丢失的分区或修复启动问题，操作起来相对简单，能快速找回丢失的分区信息。 

::: info 相关推荐
[PhotoRec](https://www.cgsecurity.org/wiki/PhotoRec_CN)：一款数据恢复软件。能从存储卡、硬盘等多种设备恢复超过 480 种格式的丢失文件，支持添加自定义文件签名，并与 TestDisk 捆绑，通过从原始数据中识别文件特征来恢复，而非依赖文件系统。 
:::
