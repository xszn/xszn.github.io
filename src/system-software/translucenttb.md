# TranslucentTB

## 介绍

<Img alt="“TranslucentTB”屏幕截图" src="/images/system-software/translucenttb/screenshot.webp" />

TranslucentTB 是一款桌面美化软件。能将任务栏设置为透明、半透明或模糊效果，支持动态模式、高级颜色定制和亚克力毛玻璃效果，极低占用内存和 CPU，让桌面更美观，与 Wallpaper Engine 等软件搭配效果更佳。

## 系统平台

Windows
