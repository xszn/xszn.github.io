# Bitwarden

## 介绍

<Img alt="“Bitwarden”屏幕截图" src="/images/system-software/bitwarden/screenshot.webp" />

Bitwarden 是一款密码管理器。其设计宗旨是帮助用户在一个高度安全的、加密的保管库中存储和管理各种敏感信息，例如网站登录凭据、安全笔记和信用卡信息。
