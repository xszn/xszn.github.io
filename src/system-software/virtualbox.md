# VirtualBox

## 介绍

<Img alt="“VirtualBox”屏幕截图" src="/images/system-software/virtualbox/screenshot.webp" />

VirtualBox 是一款虚拟机。支持创建虚拟机来运行包括 Windows、Linux、BSD 等在内的多种操作系统。还有一个显著特点是可以通过安装包含驱动和应用程序的“客户添加”包，来提升虚拟机性能。

## 系统平台

Windows、macOS、Linux

## 下载

网站：https://www.virtualbox.org/

源码：https://github.com/VirtualBox/virtualbox
