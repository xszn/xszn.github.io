# Microsoft PowerToys

## 介绍

<Img alt="“Microsoft PowerToys”屏幕截图" src="/images/system-software/microsoft-powertoys/screenshot.webp" />

Microsoft PowerToys 是一款 Windows 工具箱。专为高级用户设计的 Windows 实用工具集合，旨在提升生产力。它包含了一系列开源、免费的工具。

## 系统平台

Windows
