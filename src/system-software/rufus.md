# Rufus

## 介绍

<Img alt="“Rufus”屏幕截图" src="/images/system-software/rufus/screenshot.webp" />

Rufus 是一款启动盘制作工具。用于格式化 USB 驱动器并创建可启动的 USB，是制作启动盘（如 Windows 或 Linux 安装盘）的流行工具。
