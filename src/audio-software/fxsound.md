# FxSound

## 介绍

<Img alt="“FxSound”屏幕截图" src="/images/audio-software/fxsound/screenshot.webp" />

FxSound 是一款声音增强器。提升电脑音质。可增强音质、音量和低音效果，包含均衡器、效果和预设，方便自定义音频。

## 系统平台

Windows
