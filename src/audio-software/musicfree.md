# MusicFree

## 介绍

<Img alt="“MusicFree”屏幕截图" src="/images/audio-software/musicfree/screenshot.webp" />

MusicFree是一款音乐搜索器。通过导入自定义源，可以在线播放、下载音乐，支持多个音乐平台切换。使用前建议调整一遍软件设置，阅读一遍说明文档。

::: info 相关推荐
[MusicFree-Mobile](https://github.com/maotoumao/MusicFree)：插件化、定制化、无广告的免费音乐播放器

[MusicFree-Desktop](https://github.com/maotoumao/MusicFreeDesktop)：插件化、定制化、无广告的免费音乐播放器

**懒人订阅链接:** `https://musicfreepluginshub.2020818.xyz/plugins.json`

:::
