# AutoHotkey

## 介绍

<Img alt="“AutoHotkey”屏幕截图" src="/images/productivity-software/autohotkey/screenshot.webp" />

AutoHotkey 是一款自动化软件。主要用于通过自定义热键、宏和脚本来自动化重复性任务，从而提高效率。能模拟键盘和鼠标操作、进行窗口管理、创建简单的图形界面等，适用于从初学者到高级用户。

## 系统平台

Windows
