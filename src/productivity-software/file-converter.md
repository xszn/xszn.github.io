# File Converter

## 介绍

<Img alt="“File Converter”屏幕截图" src="/images/productivity-software/file-converter/screenshot.webp" />

File Converter 是一款文件转换软件。通过集成到 Windows 资源管理器，右键点击文件就能快速选择目标格式进行转换或压缩，支持图片、音视频、文档等多种类型，操作极为简便。

## 系统平台

Windows
