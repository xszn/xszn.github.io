# Ollama

## 介绍

<Img alt="“Ollama”屏幕截图" src="/images/productivity-software/ollama/screenshot.webp" />

Ollama 是一款本地 LLM 工具。可以在本地电脑上执行语言模型，无需服务器。快速上手使用 OpenAI gpt-oss、DeepSeek-R1、Gemma 3 和其他模型。

## 系统平台

Windows、macOS、Linux

## 下载

网站：https://ollama.com/

源码：https://github.com/ollama/ollama
