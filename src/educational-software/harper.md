# Harper

## 介绍

<Img alt="“Harper”屏幕截图" src="/images/educational-software/harper/screenshot.webp" />

Harper 是一款语法检查器。支持美式英语、英式英语、加拿大英语和澳大利亚英语，是 Grammarly 的开源替代品，完全在本地运行。
