# Anki

## 介绍

<Img alt="“Anki”屏幕截图" src="/images/educational-software/anki/screenshot.webp" />

Anki 是一款抽认卡软件。主要用于制作和学习抽认卡。该软件基于认知科学研究，利用间隔重复和主动回忆的原理来提高学习效率和长期记忆保持。

## 系统平台

Web、Android、iOS、Windows、macOS、Linux
