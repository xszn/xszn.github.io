# GeoGebra

## 介绍

<Img alt="“GeoGebra”屏幕截图" src="/images/educational-software/geogebra/screenshot.webp" />

GeoGebra 是一款动态几何软件。完美结合了几何与代数，提供了一个集几何、代数、微积分、数据、统计于一体的免费、开源平台，方便教师和学生探索数学概念。

## 系统平台

Web、Android、iOS、Windows、macOS、Linux

源码：https://github.com/geogebra/geogebra
