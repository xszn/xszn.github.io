# Playnite

## 介绍

<Img alt="“Playnite”屏幕截图" src="/images/video-game-software/playnite/screenshot.webp" />

Playnite 是一款游戏启动器。支持 Steam、Epic、GOG、EA App、Battle.net 等第三方游戏库。还包含游戏模拟功能，为游戏提供统一的界面。

## 系统平台

Windows
