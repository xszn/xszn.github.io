# Prism Launcher

## 介绍

<Img alt="“Prism Launcher”屏幕截图" src="/images/video-game-software/prism-launcher/screenshot.webp" />

Prism Launcher 是一款 Minecraft 启动器。允许创建和管理多个 Minecraft 安装，或者更准确地说是“实例”，每个实例都可以拥有各自独立的模组、资源包、设置等。

::: tip 相关推荐
[HMCL](https://hmcl.huangyuhui.net/)：一款 Minecraft 启动器。更适合国人使用的启动器，支持模组管理、游戏自定义、游戏自动安装（Forge、NeoForge、Cleanroom 等）、集成包创建、自定义界面等功能。
:::
