# ShaderGlass

## 介绍

<Img alt="“ShaderGlass”屏幕截图" src="/images/video-game-software/shaderglass/screenshot.webp" />

ShaderGlass 是一款着色器。将 GPU 着色器效果实时叠加到桌面的任何窗口或整个桌面上，实现 CRT 模拟、滤镜、锐化、模糊等各种视觉增强效果，尤其在模拟器和复古游戏方面应用广泛。

## 系统平台
