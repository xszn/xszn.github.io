## Yaak <Badge type="tip" text="桌面端" />

离线 API 客户端。支持 REST、GraphQL、gRPC、WebSocket 和 Server-Sent Events 等多种协议，提供导入功能、响应数据检查、插件扩展和多窗口操作。
