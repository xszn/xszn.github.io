# HandBrake

## 介绍

<Img alt="“HandBrake”屏幕截图" src="/images/video-software/handbrake/screenshot.webp" />

HandBrake 是一款视频转换器。将各种视频格式转换编码为常见封装格式，如 MP4、MKV 或 WebM，以提高兼容性、减小文件大小或适应不同设备。

## 系统平台

Windows、macOS、Linux
