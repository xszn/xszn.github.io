# mpv

## 介绍

<Img alt="“mpv”屏幕截图" src="/images/video-software/mpv/screenshot.webp" />

mpv 是一款媒体播放器。支持多种视频文件格式、音频和视频编解码器以及字幕类型。还提供特殊的输入 URL 类型，可以从磁盘文件以外的各种来源读取输入。

::: info 相关推荐
[mpv.net](https://github.com/mpvnet-player/mpv.net)：一款媒体播放器。具有现代化的图形用户界面。出于兼容 mpv 而设计，几乎所有 mpv 的功能都可用，因为都包含在 libmpv 中，这意味着 mpv 官方手册也适用于 mpv.net。
:::
