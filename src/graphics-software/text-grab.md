# Text Grab

## 介绍

<Img alt="“Text Grab”屏幕截图" src="/images/graphics-software/text-grab/screenshot.webp" />

Text Grab 是一款图片转文字。利用 Windows 内置的 OCR 功能抓取屏幕上的任何文本。无需在后台运行，也无需网络连接即可工作。

## 系统平台

Windows
