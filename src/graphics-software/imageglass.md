# ImageGlass

## 介绍

<Img alt="“ImageGlass”屏幕截图" src="/images/graphics-software/imageglass/screenshot.webp" />

ImageGlass 是一款图片查看器。支持超过 90 种常见图像格式，包括 WEBP、GIF、SVG、PNG、JXL、HEIC、RAW 等。还提供高级功能，满足普通用户和设计师的需求，是提升工作效率的理想工具。

## 系统平台

Windows
