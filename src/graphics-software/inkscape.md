# Inkscape

## 介绍

<Img alt="“Inkscape”屏幕截图" src="/images/graphics-software/inkscape/screenshot.webp" />

Inkscape 是一款矢量图编辑软件。功能强大，可用于创建徽标、图表、卡通、网页图形等，并以 SVG（可缩放矢量图形）为原生文件格式，与 Web 标准兼容性好。

## 系统平台

Windows、macOS、Linux
