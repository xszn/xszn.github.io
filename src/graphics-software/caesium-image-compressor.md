# Caesium Image Compressor

## 介绍

<Img alt="“Caesium Image Compressor”屏幕截图" src="/images/graphics-software/caesium-image-compressor/screenshot.webp" />

Caesium Image Compressor 是一款图像压缩软件。可存储、发送和共享数字图像，支持 JPG、PNG、WebP 和 TIFF 格式。可以在保持图像整体质量的前提下，快速减小文件大小。

## 系统平台

Windows、macOS
