# RustDesk

## 介绍

<Img alt="“RustDesk”屏幕截图" src="/images/network-software/rustdesk/screenshot.webp" />

RustDesk 是一款远程桌面软件。可以远程维护计算机和其他设备。是安全、私密和可定制的远程访问的完美 TeamViewer 替代品。

## 系统平台

Web、Android、iOS、Windows、macOS、Linux
