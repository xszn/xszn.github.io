# Buster

## 介绍

<Img alt="“Buster”屏幕截图" src="/images/network-software/buster/screenshot.webp" />

Buster 是一款验证码自动识别器。利用语音识别技术完成 reCAPTCHA 音频挑战，解决复杂的验证码问题。只需点击 reCAPTCHA 小部件底部的扩展程序按钮即可完成挑战。
