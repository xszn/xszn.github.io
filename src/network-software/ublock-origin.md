# uBlock Origin

## 介绍

<Img alt="“uBlock Origin”屏幕截图" src="/images/network-software/ublock-origin/screenshot.webp" />

uBlock Origin 是一款广告拦截软件。默认会阻止广告和网站追踪器，并供用户自行选择过滤其他内容。以 CPU 和内存使用率为主要特征。
