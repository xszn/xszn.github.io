# Joplin

## 介绍

<Img alt="“Joplin”屏幕截图" src="/images/text-software/joplin/screenshot.webp" />

Joplin 是一款笔记软件。采用 Markdown 格式，可以管理大量按笔记本分类的笔记。笔记可搜索、可复制、可添加标签，并且可以直接在应用内或通过文本编辑器进行修改。

## 系统平台

Android、iOS、Windows、macOS、Linux

## 下载

网站：https://joplinapp.org/

源码：https://github.com/laurent22/joplin
