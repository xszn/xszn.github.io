# Notepad Next

## 介绍

<Img alt="“Notepad Next”屏幕截图" src="/images/text-software/notepad-next/screenshot.webp" />

Notepad Next 是一款文本编辑器。Notepad++ 的跨平台重实现项目，允许用户在不同操作系统上获得一致的编辑体验，支持插件、主题和类似 Notepad++ 的界面。

## 系统平台

Windows、macOS、Linux
