# ImHex

## 介绍

<Img alt="“ImHex”屏幕截图" src="/images/text-software/imhex/screenshot.webp" />

ImHex 是一款十六进制编辑器。集成了数据可视化、分析和修改能力，尤其擅长二进制格式的逆向工程，提供了如自定义模板、节点式处理器、反汇编等付费软件才有的高级特性。

## 系统平台

Windows、macOS、Linux
