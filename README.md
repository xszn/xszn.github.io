# 行书指南

![行书指南](https://xszn.org/images/ogp.webp)

行书指南是一个自由及开放源代码（FOSS）的软件列表项目，推荐高质量免费与开源软件，降低用户寻找软件的时间成本。

## :bust_in_silhouette: 适用于谁

本项目适用于 Windows、Android 和 Chromium 的用户。
