2025 年 7 月软件推荐

  • Mark Dinn
  • 2025-07-28
    • 工具

2025 年 7 月软件推荐。每天一款软件，发现你的创作与工作。每天 10:00 更新。

2025 年 7 月 28 日

FileCodeBox（文件快递柜）Web

轻量级、匿名口令文件分享工具。支持多格式文件上传（最大 10MB），具备安全加密、过期时间设置、取件选择和发件记录功能。

Tabiew 桌面端

轻量级命令行表格数据查看和查询工具。支持多种文件格式（如 CSV、Parquet、JSON），拥有 Vim 风格快捷键、内置 SQL 查询、多表支持和数据可视化。

zenta 桌面端

终端专注练习工具。提供多种呼吸练习和优美动画，在编码时保持专注。

2025 年 7 月 27 日

Win11Debloat Windows

提升 Windows 11 使用体验的开源脚本。能快速删除 Windows 预装应用、关闭遥测和广告，优化系统界面和性能。

Colanode Web 桌面端

开源的本地优先协作平台。集成实时聊天、富文本编辑和文件管理，保障数据隐私，支持离线工作和多服务器多工作空间管理。

DataEase 桌面端

开源的商业智能（BI）工具。支持多操作系统和多种数据源，通过拖拽快速制作丰富图表与大屏。

2025 年 7 月 26 日

RustScan 桌面端

快速端口扫描工具。能在 3 秒内扫描所有端口，支持多种脚本语言扩展和自适应学习优化，自动配合 Nmap 深入分析。

Semgrep Web

快速、开源的静态代码分析工具。支持 30 多种编程语言，能在开发环境、代码提交前和 CI/CD 流程中自动扫描代码，发现安全和质量问题。

OpenBB Web

开源的金融分析平台。支持多种资产类别的数据查询与分析，提供命令行和企业级可视化工作台。

2025 年 7 月 25 日

Cursor Free Trial Reset Tool CLI

Cursor 免费试用重置工具。通过自动修改 Cursor 软件的机器码，绕过试用限制，继续免费使用 Cursor 的高级功能。

Kirara AI 桌面端

多平台、多大语言模型支持的可定制聊天机器人。集成图片发送、关键词回复、多账号管理及语音对话等功能。

Frappe HR Web

开源的综合性人力资源和薪资管理软件。多功能模块、高度定制和安全可靠的解决方案，简化员工管理及薪资流程。

2025 年 7 月 24 日

Cursor Free VIP 桌面端

专为 Cursor AI 设计的辅助工具。能自动重置机器 ID，绕过试用限制，继续免费使用高级功能。

PakePlus Web 桌面端

开源轻量级跨平台打包工具。可快速将网页、Vue/React 项目打包成小于 5MB 的高性能桌面和移动应用。

Jujutsu 桌面端

兼容 Git 的现代分布式版本控制系统。特点是简化工作流程、自动管理提交状态、支持冲突自动解决和高效并发复制。

2025 年 7 月 23 日

Anki Web 移动端 桌面端

免费开源的智能记忆卡片软件。通过科学的间隔重复算法，根据你对知识点的掌握情况自动安排复习时间。

Trippy 桌面端

跨平台网络诊断工具。支持多种协议和高度自定义的跟踪选项，帮助你快速精准地分析和定位网络问题。

afrog 桌面端

开源的高性能漏洞扫描工具。内置多种漏洞检测PoC（如 CVE、默认密码、信息泄露等），帮助你快速发现并修复漏洞。

2025 年 7 月 22 日

NextChat Web 移动端 桌面端

开源 AI 聊天工具。支持多种大模型，无需订阅，注重隐私保护，提供免费一键部署和跨平台使用。

Worklenz Web

开源且功能全面的项目管理工具。帮助团队高效管理任务、进度和协作，支持可视化跟踪、时间管理和资源分配。

Dashy Web

开源且高度可定制的自托管仪表盘应用。支持多主题、多语言和丰富的小部件，方便集中管理和实时监控自托管服务。

2025 年 7 月 21 日

Sim Studio Web

开源的 AI 代理构建平台，有着类似 Figma 的可视化拖拽界面。

Maybe Web

开源的个人财务管理平台。支持多账户、多货币同步，提供交易分类、预算工具和财务分析。

Magisk Android

开源的 Android 定制工具，支持在不解锁 Bootloader 的情况下深度定制系统，现如今的手机功能足够强大，喜欢折腾的可以玩玩。

2025 年 7 月 20 日

shadPS4 桌面端

为电脑用户开发的 PlayStation 4 模拟器。支持的 Windows、Linux 和 macOS 等系统。由于还在开发中，稳定性和兼容性有限，能运行的游戏数量不多。

Harden Windows Security Windows

增强 Windows 安全性的开源项目。提供全面的安全措施，包括组策略配置、PowerShell 命令、设备和网络保护等。

Remote-friendly companies Web

这个网站整理了一份支持远程办公的科技公司名单，覆盖全球各地。可以按地区、公司名称或官网进行搜索，方便找到合适的远程职位。

2025 年 7 月 19 日

RunCat 365 Windows

一款桌面小工具。在你的 Windows 任务栏上显示一个可爱的奔跑猫动画。

LTX-Video Web

首个基于 DiT 的视频生成模型，能够实时生成高质量视频。它能以 1216×704 的分辨率生成每秒 30 帧的视频，速度比观看这些视频还要快。

LinkAI Web

一站式 AI 智能体平台。聚合文本、语音、图像等多模态模型，提供 RAG 知识库、数据库、插件、工作流等 Agent 核心能力，并支持零代码接入网站、企业微信、公众号、微信客服、钉钉、飞书等多种应用渠道。

2025 年 7 月 18 日

Maigret 桌面端

开源的网络信息收集工具。可以通过输入一个用户名，帮你在上千个网站上查找同名账户，然后收集这些网站上公开的个人信息，整理成一个人的档案。

Helix 桌面端

后现代模态文本编辑器。类似 Vim 的模态编辑，使用不同的编辑模式（如普通模式、插入模式），实现高效文本操作。

Gitleaks 桌面端

开源的 stdin 秘密扫描器。用于检测 git 仓库、文件中的密码、API 密钥和令牌等机密信息，以及任何你想通过 stdin 输入的信息。

2025 年 7 月 17 日

VpnHood Client Android Windows

适用于普通用户和专家的无法检测的 VPN。绕过高级防火墙和规避深度数据包检查的解决方案。

DocsGPT Web

开源 genAI 工具。可帮助用户从任何知识来源获得可靠答案，同时避免产生幻觉。

Zotero Style 扩展程序

Zotero 美化插件。增加提升效率的功能，如阅读进度可视化、标签管理、快速过滤等。

2025 年 7 月 16 日

NumPad Web

功能强大的记事本计算器。支持单位、变量、日期、时间等。

SpeedyNote（极疾记）桌面端

专注于压感墨水输入的跨平台高效笔记应用。能够在低于 1GHz 的 Intel Atom 处理器上以 360Hz 刷新率运行压感墨迹。

DLSS Swapper Windows

能够让您方便地下载、管理与切换 DLSS、FSR 和 XeSS 的 DLL 实用工具。使您能够在不更新游戏的情況下更新或是回退的版本。

2025 年 7 月 15 日

WebVM Web

Web 虚拟机。可在浏览器中运行的 Linux 虚拟机。

KitchenOwl Web 移动端 桌面端

智能的自托管购物清单和食谱管理器。可以在购物前轻松将商品添加到购物清单，还可以创建食谱并获得烹饪建议。

phpIPAM Web

开源的 Web IP 地址管理器。其目标是提供轻量、现代且实用的 IP 地址管理功能。

2025 年 7 月 14 日

NoteGen 移动端 桌面端

跨平台的 Markdown AI 笔记软件。使用 AI 建立记录和写作的桥梁，将碎片化知识整理成一篇可读的笔记。

Lazyeat 桌面端

手势隔空控制软件。对着摄像头比划手势就能暂停视频/全屏/切换视频。

ChatGPT 中文调教指南 Web

它能够生成类似于人类写作的文本。您只需要给出提示或提出问题，它就可以生成你想要的东西。

2025 年 7 月 13 日

浸入式学语言助手 扩展程序

基于“可理解输入”理论的浏览器扩展。通过 AI 智能翻译将网页内容转化为个性化学习材料，帮助你在日常网页浏览中自然地学习语言。

OCRmyPDF 桌面端

PDF 文档命令行工具。为扫描的 PDF 文件添加 OCR 文本层，以便搜索或复制粘贴。

uBlock Origin Lite 扩展程序

一个不需要权限的内容屏蔽工具。安装即可屏蔽广告、跟踪器、挖矿脚本等网页内容。

2025 年 7 月 12 日

Agentic Document Extraction Web

文档提取工具。可从 PDF 和图像等非结构化文档中提取结构化数据。

Linux Command Web

Linux 命令大全搜索工具。内容包含 Linux 命令手册、详解、学习、搜集。

How To Cook Web

程序员做饭指南。用更清晰精准的描述来整理常见菜的做法，以方便程序员在家做饭。

2025 年 7 月 11 日

Quick Reference Web

为开发人员分享快速参考备忘清单（速查表）。帮助自己更好地使用相关技术。

fzf 桌面端

通用的命令行模糊查找工具。可过滤文件、命令历史记录、进程、主机名、书签、git 提交等任何类型的列表。

Fscan 桌面端

内网综合扫描工具。包含端口扫描、服务识别、网络资产探测、密码爆破等功能。

2025 年 7 月 10 日

res-downloader（爱享素材下载器） 桌面端

基于 Go + Wails 的跨平台资源下载工具，简洁易用，支持多种资源嗅探与下载。

ToastFish Windows

利用 Windows 通知栏背单词的软件。可以让你在上班、上课等恶劣环境下安全隐蔽地背单词。

ProxyPin 桌面端 移动端

开源免费抓包工具。你可以使用它来拦截、检查和重写 HTTP(S) 流量，支持全平台系统。

Trail Sense Android

野外跋涉协助工具。专为徒步旅行、背包旅行、露营和寻宝而设计，利用手机的传感器来协助野外跋涉或生存，可离线使用。

OpenList 桌面端

AList 的新分支。支持多种存储、网页浏览和 WebDAV 的文件列表程序。

defendnot Windows

禁用 Windows Defender。通过 WSC API 方法实现，支持脚本、EXE 程序使用。

Hover Zoom+ 扩展程序

用于在鼠标悬停时缩放图像的扩展程序。将你的鼠标指针指向任何被支持的网站图片上，这个扩展会自动把图片/视频放大至全尺寸。

FlashDim Android

适用于 Android 13+ 的现代手电筒应用。具有调暗功能，可以控制手电筒的多个亮度级别。
