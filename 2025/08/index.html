2025 年 8 月软件推荐

  • Mark Dinn
  • 2025-08-12
    • 工具

2025 年 8 月软件推荐。每天一款软件，发现你的创作与工作。每天 10:00 更新。

2025 年 8 月 12 日

Bloxstrap Windows

第三方 Roblox 启动器。提供多种实用功能和安全认证，让玩家能更方便地个性化游戏体验。

PocketBase Web 桌面端

轻量开源的后端工具。集成数据库、用户认证、文件存储和管理面板，支持实时数据和多平台开发。

KernelSU Android

安卓设备 root 权限管理工具。通过内核级别的控制和模块化设计，实现安全可靠且灵活的权限管理。

2025 年 8 月 11 日

yt-dlp 桌面端

音视频下载工具。支持从 1000 多个网站下载和自定义处理音视频内容，在命令行上操作，可以下载 YouTube 上的视频。

GitMCP Web

开源免费的远程 MCP 服务器工具。通过将 GitHub 项目转为文档中心，让 AI 更精准理解代码和文档。

Umami Web

现代网站数据分析工具。是 Google Analytics 的优秀替代品，支持实时监控、多种数据库和便捷部署。

2025 年 8 月 10 日

GPT4All 桌面端

开源本地运行大型语言模型工具。提供多种模型下载，即下即用，支持多平台和多硬件，高效且可定制的离线聊天体验。

bitchat 移动端

去中心化社交聊天应用。利用蓝牙网状网络实现无互联网环境下的加密离线通讯，保护隐私并支持多平台使用。

Hoppscotch Web 桌面端

开源 API 开发工具。Postman、Insomnia 的替代品，支持多种协议和团队协作，提供网页和桌面应用版本。

2025 年 8 月 9 日

Mate Engine Windows

桌面软件 Desktop Mate 的替代品。免费开源，支持自定义 VRM 模型和插件，提供流畅动画和多种交互功能。

JupyterLite Web

轻量级交互式计算环境。在浏览器中运行 Python 和 JavaScript 代码，支持多核管理及丰富的可视化。

二哥的 Java 进阶之路 Web

通俗易懂、内容全面的 Java 学习资源合集。涵盖基础到高级知识和实战项目，可以系统性的提升技能。

2025 年 8 月 8 日

铜钟 Web

免费音乐网站。有段时间停运过，现在复活了！有着丰富资源和简洁界面，方便搜索、播放、保存歌曲及管理歌单。

lazygit 桌面端

Git 终端 UI 工具。简化 Git 操作，提供友好界面和丰富功能，在终端也能使用 UI 操作 Git 命令。

Simple Live 移动端 桌面端

跨平台看直播 APP。支持多个主流平台和设备，作者不提供安装包，动手能力强的朋友可以自己去编译打包。

2025 年 8 月 7 日

Drawnix Web

开源免费在线白板工具。支持思维导图、流程图和自由绘画，具备丰富编辑功能、移动适配及多种导出格式。

Hetty 桌面端

开源的 HTTP 工具包。专为安全研究和漏洞赏金设计，提供中间人代理、请求编辑和审查等功能。

IntelliJ IDEA 桌面端

IDEA 的社区开源版本。集成 AI 辅助功能、支持最新技术和模块化开发，用惯了 VS Code 的可以去尝试下。

2025 年 8 月 6 日

RevokeMsgPatcher Windows

防撤回补丁工具。支持 PC 版微信、QQ 和 TIM 最新版本，通过修改关键文件实现多开和防撤回功能。

Vibe 桌面端

音视频转文字转录工具。支持多语言、离线运行、保护隐私、功能丰富且易用，兼容多平台并持续更新。

Hashcat 桌面端

号称世界上最快的密码破解工具。支持多种攻击模式和超过 450 种哈希算法，利用 CPU、GPU 加速和分布式功能。

2025 年 8 月 5 日

AltSnap Windows

Windows 窗口管理工具。通过按住 Alt 键并用鼠标点击任意位置，轻松移动和调整窗口大小。

ScriptCat（脚本猫）扩展程序

可以执行用户脚本的浏览器扩展。兼容油猴脚本，支持后台持续运行、自定义网页功能、广告屏蔽和自动化任务执行。

FossFLOW Web

网页等距制图工具。支持离线制作和自动保存美观的等角投影 3D 图表，注重隐私保护并支持导入导出与原生 PWA 安装。

2025 年 8 月 4 日

opencode 桌面端

完全开源 AI 编码助手。采用客户端/服务器架构，支持多种 AI 模型与本地模型，提供原生终端用户界面和多会话管理。

SkyReels Web

AI 短剧创作平台。实现无限时长、高质量电影级视频生成，具备故事创作、图像转视频、摄像导演和多主体一致性等功能。

Jellyfin 移动端 桌面端

完全免费开源媒体系统。可以通过自建服务器管理和流式传输电影、音乐、电视等多种媒体内容。

2025 年 8 月 3 日

Tandoor Recipes Web

食谱管理和分享工具。支持食谱整理、膳食计划、购物清单生成、协作编辑及多平台导入。

ESP32-BlueJammer Flasher Web

多模式 2.4GHz 无线干扰器。能干扰蓝牙、BLE、WiFi 和遥控设备通信。

SUMO 桌面端

微观交通仿真软件。支持多种交通方式和大型道路网络的高性能仿真，拥有丰富的场景创建及控制工具。

2025 年 8 月 2 日

VideoLingo Web

一站式视频翻译、本地化和配音工具。利用 AI 技术自动生成符合 Netflix 标准的高质量单行字幕和自然配音。

PlayCanvas Web

开源的基于网页的图形创作平台。提供浏览器内的实时协作 WebGL 编辑器，支持多人共同开发 HTML5 游戏和视觉内容。

Dyad 桌面端

免费开源的本地 AI 应用构建工具。无需编码通过可视化界面快速创建全栈 AI 应用，支持使用自有 API 密钥连接多种顶级 AI 模型。

2025 年 8 月 1 日

DuckStation Android 桌面端

开源、跨平台的 PlayStation 1 模拟器。支持多种 CPU 架构和图形接口，适合在低端设备上流畅运行各类经典游戏。

Linkwarden Web

开源且自托管的协作书签管理工具。自动保存网页截图、PDF 和内容备份，支持多层次组织、注释、高效搜索和多人协作。

FlutterUnit Web 移动端 桌面端

跨平台的 Flutter 组件库和测试框架应用。收录 300 多个组件，提供交互演示、代码示例和个性化配置。
