Clash Verge Rev：代理软件推荐，解锁网络自由神器

  • Mark Dinn
  • 2025-03-23
    • 工具

Clash Verge 是一款界面美观、操作简单的科学上网工具，支持 Windows、macOS 和 Linux 系统，内置 Clash.Meta 内核并可切换 Alpha 版本。适合新手和老手使用，Windows 用户建议用 x64 版本，Win7 不支持。软件自带 Webview2 环境，GitHub 提供正式版和测试版下载，还有详细安装和使用教程，支持多种代理协议，方便快速配置和使用。

使用教程

1. 导入订阅

订阅是节点信息，打开 Clash Verge 左侧 订阅 用 URL 导入链接，导入后配置文件显示列表，点击即可使用。

2. 选择节点和模式

点左侧的 代理，在代理组里选一个节点，点一下就能切换。

在右上角的模式里选：

  • 规则：按规则走代理，适合日常上网。
  • 全局：所有流量都走代理。
  • 直连：所有流量都不走代理。

3. 开启代理

点左侧的 设置，开启代理或 Tun 模式：

  • 系统代理：Clash Verge 会自动设置系统代理。
  • Tun 模式：开关，适合游戏或命令行工具等复杂场景。

Clash Verge 是一个简单又好用的科学上网工具。用它上网更自由，也更安全。如果有问题，可以去 GitHub 上找答案或提建议。
